## 23:45 更新 — Claude Code 终于跑起来了

### Claude Code --print 不支持 Claude Max OAuth
- `claude --print` 非交互模式不走 OAuth，只认 API key
- Claude Max 订阅登录是 OAuth，所以 `--print` 模式永远报 Invalid API key
- **解决方案：用 PTY 交互模式跑 Claude Code**（`exec pty:true` + `process paste/send-keys`）
- 发现 OpenClaw 环境里有 ANTHROPIC_API_KEY 环境变量，Claude Code 检测到了并使用
